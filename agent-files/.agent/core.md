# Core Loop

1. Resolve the oldest open interrupt.
2. Otherwise resume the next incomplete plan step.
3. Name the active step and acceptance before editing.
4. If pausing, record `Pauses:` and `Resume From:`.
5. Log issues immediately and update lessons after.
6. Verify acceptance before marking anything complete.
7. Keep required records current before moving on.
8. Report assumptions and decisions after each completed step.
9. Stop and surface conflicts or ambiguity.
