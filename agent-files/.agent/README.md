# Agent Workspace Template

Copy this folder to the target project root as `.agent/`.

## Required files

- `core.md`, `current.md`: workflow control
- `requirements.md`, `plan.md`: scope and execution
- `changelog.md`: append-only history
- `decisions.md`, `issues.md`, `lessons-learned.md`: quality trail
- `context.md`, `conventions.md`, `scratch.md`, `rotate_changelog.md`: support files
